## BACKEND
 run the development server:

```bash
# step 1:
- config host && port in file config.json

# step 2:
- run npx sequelize db:migrate or import file sql from folder database in the project

# step 3:
- cd /backend

# step 4:
- npm i

# step 5
- npm start

import file json [postman_request_api.json] from folder postman_json to postman   
```
